### 2ï¸âƒ£ Iterative Version (Same Meaning, Efficient Process)

```js
function eC(days) {
  if (days === 0) return 0;
  if (days === 1) return 1;
  if (days === 2) return 2;

  // f(2), f(1), f(0)
  let prev1 = 2;
  let prev2 = 1;
  let prev3 = 0;

  for (let i = 3; i <= days; i++) {
    const todayEnergy = prev1 + 2 * prev2 + 3 * prev3;

    // slide the state forward
    prev3 = prev2;
    prev2 = prev1;
    prev1 = todayEnergy;
  }

  return prev1;
}
```

    Iteration should **carry state forward**, not recompute past values.
